# スライドショー セットアップ手順書（Windows）

この手順書は **Windows 環境** でスライドショーを初めて利用する方向けです。  
上から順に実施することで、ローカル確認 → GitHub Pages 公開まで完了します。

---

## 前提条件

以下がインストールされていることを確認してください。

- **Git for Windows** — https://git-scm.com/download/win
- **Web ブラウザ**（Chrome / Edge / Firefox いずれか）

---

## STEP 1 ： リポジトリをクローンする

1. エクスプローラーで作業フォルダを開く（例：`C:\dev`）
2. アドレスバーに `cmd` と入力して Enter → コマンドプロンプトが開く
3. 以下を実行する

```
git clone https://github.com/<ユーザー名>/<リポジトリ名>.git
cd <リポジトリ名>
```

> クローン後、フォルダ内に `index.html` や `generate-list.bat` があれば成功です。

---

## STEP 2 ： 画像を追加する

1. クローンしたフォルダ内の `images\` フォルダを開く
2. 表示したい画像ファイルをコピーして貼り付ける

**対応形式：** `jpg` / `jpeg` / `png` / `gif` / `webp` / `bmp`

> ファイル名に日本語・スペースが含まれていても動作しますが、  
> 英数字とアンダースコアのみ（例：`photo_001.jpg`）が安全です。

---

## STEP 3 ： 画像リストを生成する

1. クローンしたフォルダ内の **`generate-list.bat`** をダブルクリックする
2. コマンドプロンプトが開き、以下のような出力が表示されれば成功

```
images-list.js を生成中...
Done: XX image(s) written to images-list.js
続行するには何かキーを押してください . . .
```

3. 何かキーを押してウィンドウを閉じる

> `images-list.js` が更新されます。このファイルは自動生成のため直接編集しないでください。

---

## STEP 4 ： ブラウザで動作確認する

1. クローンしたフォルダ内の **`index.html`** をダブルクリックする
2. ブラウザが起動し、スライドショーが自動再生されることを確認する

**確認ポイント：**
- 画像が表示されてフェードで切り替わる
- 画面下部にコントロールバーが表示される（マウスを動かすと出てくる）
- 左下のカウンターに `1 / XX` と枚数が表示される

**画像が表示されない場合：**
- `images\` フォルダに画像が入っているか確認する
- STEP 3 を再度実施して `images-list.js` が更新されているか確認する
- ブラウザを再読み込み（`F5`）する

---

## 画像を追加・更新するとき（2回目以降）

```
1. images\ に画像ファイルを追加
2. generate-list.bat をダブルクリック
3. index.html をブラウザで確認
```

---

## 操作方法

| 操作 | 動作 |
|---|---|
| スペースキー | 一時停止 / 再開 |
| ⏸ / ▶ ボタン | 一時停止 / 再開 |
| ⛶ ボタン | フルスクリーン切り替え |
| 進捗バークリック | その位置の画像にジャンプ |
| マウスを動かす | コントロールバーを表示 |
